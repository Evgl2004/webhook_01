## Разработка системы анализа гостей и управления рассылкой (САГУР) гостями на базе Django с данными из iikoCard.

### Серверная часть проекта (backend) предполагает реализацию следующего функционала:

- Возможность получать и хранить уведомления со сторонних сервисов без выполнения авторизации.

### Использованные технологии:
- Python;
- Django;
- Django REST framework;
- Postgres;
- CORS;
- Docker;
- Docker Compose.
